# Big O Zaman Karmaşıklığı Analizleri

Bu dosyada 7 farklı algoritmanın zaman karmaşıklığı analizleri yer almaktadır.

| Algoritma | Zaman Karmaşıklığı (Big O) | Açıklama |
| :--- | :--- | :--- |
| **Algo1** | $O(n)$ | Lineer döngü |
| **Algo2** | $O(n^2)$ | İç içe (nested) döngü |
| **Algo3** | $O(\log_2 n)$ | İkiye katlanarak artan indeks |
| **Algo4** | $O(\log_5 n)$ | Beşe bölünerek azalan indeks |
| **Algo5** | $O(\log_5 n^3)$ | Kübik sınırda logaritmik artış |
| **Algo6** | $O(\sqrt{n})$ | Karekök sınırına kadar artış |
| **Algo7** | $O(\sqrt{n})$ | Aritmetik artışlı seri toplamı |
